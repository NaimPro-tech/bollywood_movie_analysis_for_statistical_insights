## Bollywood Movie Analysis
This project is for statistical analysis and dig out valuable insights from movie dataset.

## About Dataset:
Dataset contains movie information that release on 2013-2015.
Feature:
 1   Release Date
 2   MovieName    
 3   ReleaseTime  
 4   Genre        
 5   Budget        
 6   BoxOfficeCollection
 7   YoutubeViews         
 8   YoutubeLikes         
 9   YoutubeDislikes
 
Problem Analysis:
So we can find many useful insights like ROI(Return on Investment), which season is more profitable. Is theres between correlation with youtube preferences, etc.
